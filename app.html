FAQ

What is this?

As a subtitle translator, we do alot of work on our translations. But always there is a posibility for mistakes. That's the main goal of this software. Looking up for these common mistakes, let you relieve from your burdens.

Okay it's cool. But which types of subtitle I can get reports for?

I'm glad you asked. There are only two: SubRip (aka .srt) and SubStation Alpha (aka .ass or .ssa).

You did write this software from scratch?

Yep, I did indeed.

Can I contribute in any way?

If you wanna do something we can say contribution, just go for it, till end, seriously.Also don't forget you should do it via GitHub ...

I found bugs. What should I do?

I'll be very pleased if you post bugs on GitHub.

What these abbreviations stand for?

Characters Per Second (CPS): Number of characters needed to read within one second.

Characters Per Line (CPS): Number of characters for longest line.

Spelling Errors (SER): Words which have spelling mistakes.

Spaced N Tags (SNT) (SSA Only): Any N tag with starting or finishing with space.

Invalid Tags (ITA): Invalid tags or corrected version of row's invalid HTML.

Negative Durations (NDU): Negative durations based on end time minus begin time.

Unsupported Newlines (UNE) (SRT Only): Not supported \n and \N newline characters.

Overlapping Lines (OLI) (SRT Only): Lines which have overlapped duration range.

Unnecessary Spaces (USP): Two or more trailing spaces.
